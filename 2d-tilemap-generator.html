2D Tilemap Generator

This tool can be used to create a randomly generated tilemap for 2D.

Dimensions

First we specify how many tiles we want, for the width, and height.

Initial seed

The tilemap is usually first filled with tiles that are on/off based on a certain probability. This forms as a basis for later algorithms to work with. There are different types of methods that can be used to create the initial conditions. The simplest case is based on a uniformly distributed probability for the entire tilemap. Initial probability: 0 - 100

Generations

Based on the initial conditions, we apply a few iterations of the game of life algorithm.

Smoothening

Do some smoothening.

Rule tiles

Generate rule tiles.

Distance map

Generate a distance map.
